---
title: Custom Handlers
description: Teach the generator how to convert your own types into OpenAPI schemas.
---

When the generator encounters a TypeScript type, it tries each registered handler in order. The first match wins; otherwise it falls back to a structural conversion. The built-in handlers cover `luxon.DateTime`, `MultipartFile` and `LucidRow`. You can add more for your own types via `generator.handlers` in `config/openapi.ts`.

::tip
For app-level classes you control, a [`toOpenAPI`](/guide/schemas#toopenapi-tojson-overrides) method is usually simpler than a handler. Reserve custom handlers for third-party types you cannot modify (ORM rows, branded primitives, framework primitives).
::

## Handler Shape

```ts
import type { Handler } from '@outloud/adonis-openapi-generator'

const handler: Handler = {
  // match by display text (string or RegExp)
  text: /^luxon\.DateTime<[^>]+>$/,

  // match by source path (string or RegExp tested against the symbol's import path)
  path: /\/@adonisjs\/bodyparser/,

  // match by symbol name (string or RegExp)
  name: 'MultipartFile',

  // custom predicate (runs after text/path/name)
  test: (type, context) => type.isClass(),

  // produce the schema
  schema: (type, context) => ({ type: 'string' }),
}
```

A handler is considered a match when all of the supplied filters (`text`, `path`, `name`, `test`) match. Omitted filters are ignored.

## Registering Handlers

Append your handlers to `generator.handlers` in `config/openapi.ts`. They run **before** the built-in handlers, so you can override default behavior (e.g. emit `DateTime` as `date-time` instead of plain `string`).

```ts
// config/openapi.ts
import { defineConfig } from '@outloud/adonis-openapi'
import type { Handler } from '@outloud/adonis-openapi-generator'

const decimalHandler: Handler = {
  path: /\/decimal\.js/,
  name: 'Decimal',
  schema: () => ({ type: 'string', pattern: '^-?\\d+(\\.\\d+)?$' }),
}

const dateTimeHandler: Handler = {
  text: /^luxon\.DateTime<[^>]+>$/,
  schema: () => ({ type: 'string', format: 'date-time' }),
}

export default defineConfig({
  // ...
  generator: {
    resolve: import.meta.resolve,
    routes: true,
    handlers: [dateTimeHandler, decimalHandler],
  },
})
```

## Built-in Handlers

```ts
import { LucidHandler } from '@outloud/adonis-openapi-generator'

export const defaultHandlers = [
  { text: /^luxon\.DateTime<[^>]+>$/, schema: () => ({ type: 'string' }) },
  { path: /\/@adonisjs\/bodyparser/, name: 'MultipartFile', schema: () => ({ type: 'string', format: 'binary' }) },
  new LucidHandler(),
]
```

## Writing a Handler

For a class-based handler with internal logic, implement the interface directly:

```ts
import type { Handler, SchemaObject } from '@outloud/adonis-openapi-generator'

class MoneyHandler implements Handler {
  test(type) {
    return type.getSymbol()?.getName() === 'Money'
  }

  schema(): SchemaObject {
    return {
      type: 'object',
      properties: {
        amount: { type: 'number' },
        currency: { type: 'string' },
      },
      required: ['amount', 'currency'],
    }
  }
}
```

## Schema Helpers

The `context` argument exposes the generator state, useful for recursive schemas:

```ts
schema: async (type, context) => {
  const elementType = type.getTypeArguments()[0]
  return {
    type: 'array',
    items: await context.toSchema(elementType),
  }
}
```

`context.toSchema(type)` resolves any TypeScript type using the full handler pipeline, including recursion guards (`maxDepth = 10`).
